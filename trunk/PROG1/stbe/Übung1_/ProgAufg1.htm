Aufgabe 1 (Präsenzaufgabe)

"Entwickeln" Sie eine HelloWorld-Applikation.

  1. Tippen Sie zu diesem Zweck den Quellcode (vgl. Vorlesungsskript) in einem Editor (z. B. Wordpad) ein, übersetzen Sie den Quellcode und führen Sie den Bytecode aus.
    Tun Sie das alles in einem für die Aufgabe 1 angelegtem Verzeichnis und verwenden Sie die Tools des JDK in einer DOS-Box bzw. einer Unix-Shell.
  2. Ändern Sie nun die Ausgabe des HelloWorld-Programms so ab, dass das Programm Sie mit Ihrem Vornamen begrüßt.
  3. Bauen Sie absichtlich Fehler in Ihr Quellprogramm ein und beobachten Sie, wie der Compiler reagiert.
  4. Legen Sie nun in Ihrem Verzeichnis für die Aufgabe 1 Unterverzeichnisse mit Namen src und bin an.
    Verschieben Sie den Quellcode Ihres Programms in das src-Verzeichnis.
    Rufen Sie den Compiler mit einer entsprechenden Option so auf, dass der Byte-Code im bin-Verzeichnis landet.
    Starten Sie den Interpreter erneut. In welchem Verzeichnis funktioniert der Programmaufruf?
  5. Was müssen Sie tun, damit Sie das Programm in jedem beliebigen Verzeichnis ausführen können?
  6. Falls noch Zeit bleibt:
    Tippen Sie HelloWorld auch als Applet ein, zusammen mit einer entsprechenden Html-Seite.
    Bringen Sie das Applet im Browser oder Applet-Viewer zum Ablauf.

Abnahme: in der Woche vom 12. - 16. 10.
